## :star:面试题34. 二叉树中和为某一值的路径
### 前序遍历（dfs） + 参数扩展 + 回溯
* [回溯讲解](https://leetcode-cn.com/problems/permutations/solution/hui-su-suan-fa-python-dai-ma-java-dai-ma-by-liweiw/)
```JAVA
class Solution {
    public List<List<Integer>> pathSum(TreeNode root, int target) {
        //使用一个动态数组保存所有可能的全排列
        List<List<Integer>> res = new ArrayList<>();
        //路径
        List<Integer> path = new ArrayList<>();  
        //讲path和res作为参数一起传递（参数扩展）
        dfs(root, target, path, res);
        return res;
    }

    private void dfs(TreeNode root, int lastLeft, List<Integer> path, List<List<Integer>>res){
        //递归终止条件
        if (root == null){
            return;
        }
        if (root.val == lastLeft && root.left == null && root.right == null){
            path.add(root.val);
            //注意：若直接res.add(path)，传递的是path的值，即一块内存地址，后续path会变化。所以应该在这个地方复制一个path传入res
            res.add(new  LinkedList<>(path));
            //回溯：恢复现场
            path.remove(path.size() - 1);
            return;
        }
    path.add(root.val);
    dfs(root.left, lastLeft-root.val, path, res);
    dfs(root.right, lastLeft-root.val, path, res);
    ////回溯：恢复现场
    path.remove(path.size() - 1);
    }
}
```
## 1372. 二叉树中的最长交错路径
```JAVA
class Solution {
    int res = 0;
    public int longestZigZag(TreeNode root) {
        //往左记为1
        dfs(root, 1, 0);
        //往右记为-1
        dfs(root, -1, 0);
        return res;
    }
    
    private void dfs(TreeNode root, int dir, int depth){
        if (root == null){
            return;
        }
        //当前最大的距离
        res = Math.max(depth, res);
        //dir == 1，表示当前节点是父节点的左节点
        if (dir == 1){
            //继续往左走则归1
            dfs(root.left, 1, 1);
            //往右走+1
            dfs(root.right, -1, depth + 1);
        }else{
            dfs(root.left, 1, depth + 1);
            dfs(root.right, -1, 1);
        }
    }
}
```
## 面试题 04.12. 求和路径
### 双递归
* 与第一题的区别：路径不一定非得从二叉树的根节点或叶节点开始或结束，且不需要输出具体路径
```JAVA
class Solution {
    //全局变量记录符合条件的情况
    int res = 0;
    public int pathSum(TreeNode root, int sum) {
        if (root == null){
            return res;
        }
        //第一个递归dfs：解决当前root所有满足条件的情况
        dfs(root, sum);
        //第二个递归：取遍所有的点，以达到从任意节点开始
        pathSum(root.left, sum);
        pathSum(root.right, sum);
        return res;
    }

    private void dfs(TreeNode root, int sum){
        if (root == null){
            return;
        }
        if (root.val == sum){
            res = res + 1;
        }
        dfs(root.left, sum - root.val);
        dfs(root.right, sum - root.val);
    }
}
```    
## 563. 二叉树的坡度
### 方法一：双递归
* 效率很低，每次都重复计算res
```JAVA
class Solution {
    int res = 0;
    public int findTilt(TreeNode root) {
        if (root == null){
            return 0;
        }
        res = res + Math.abs(sum(root.left) - sum(root.right));
        //以每个节点为根计算坡度
        findTilt(root.left);
        findTilt(root.right);
        return res;
    }
    //计算以root为根的树所有节点之和
    private int sum (TreeNode root){
        if (root == null){
            return 0;
        }
        return root.val + sum(root.right) + sum(root.left);
    }
}
```
### 方法二
```JAVA
public class Solution {
    int tilt=0;
    public int findTilt(TreeNode root) {
        traverse(root);
        return tilt;
    }
    public int traverse(TreeNode root)
    {
        if(root==null )
            return 0;
        //左子树所有节点值
        int left=traverse(root.left);
        //右子树所有节点值
        int right=traverse(root.right);
        tilt+=Math.abs(left-right);
        return left+right+root.val;
    }
}
```

## 116. 填充每个节点的下一个右侧节点指针
### 修改类
* 用带层的 BFS 模板
```JAVA
class Solution {
    public Node connect(Node root) {
        if (root == null){
            return null;
        }
        Queue<Node> queue = new LinkedList<Node>();
        queue.add(root);
        int steps = 0;
        while(!queue.isEmpty()){
            int size = queue.size();
            Node rightNode = null;
            for (int i = 0; i < size; i ++){
                Node node = queue.peek();
                queue.remove();
                if (i == 0){
                    node.next = null;
                }else{
                    node.next = rightNode;
                }
                rightNode = node;
                if (node.right != null){
                    queue.add(node.right);
                }
                if (node.left != null){
                    queue.add(node.left);
                }
            }
            steps ++;
        }
        return root;
    }
}
```
### :star:带层的 BFS 模板（迭代）
```JAVA
class Solution {
    public Node connect(Node root) {
        if (root == null){
            return null;
        }
        //使用队列，而不是数组。因为数组从头部删除元素的时间复杂度为 N，队列的底层实现其实是链表。
        Queue<Node> queue = new LinkedList<Node>();
        queue.add(root);
        //记录层数
        int steps = 0;
        //需要返回的节点
        List<Integer> res = new ArrayList<>();  
        while(!queue.isEmpty()){
            int size = queue.size();
            Node rightNode = null;
            //遍历当前层的所有节点
            for (int i = 0; i < size; i ++){
                Node node = queue.peek();
                queue.remove();
                if (题目条件){
                    res.add(node);
                }
                if (node.right != null){
                    queue.add(node.right);
                }
                if (node.left != null){
                    queue.add(node.left);
                }
            }
            //遍历完当前层所有的节点后 steps + 1
            steps ++;
        }
        return res;
    }
}
```
## 814. 二叉树剪枝
### 修改类：dfs + 虚拟节点
```JAVA
class Solution {
    public TreeNode pruneTree(TreeNode root) {
        //因为根节点可能被移除，所以用到虚拟节点
        TreeNode res = new TreeNode(-1);
        res.left = root;
        dfs(res);
        return res.left;
    }
    //将子树和为 0 的节点移除
    private int dfs(TreeNode root){
        if (root == null){
            return 0;
        }
        int left = dfs(root.left);
        int right = dfs(root.right);
        if (left == 0){
            root.left = null;
        }
        if (right == 0){
            root.right = null;
        }
        return root.val + left + right;
    }
}
```     
## 1325. 删除给定值的叶子节点
### 后序遍历
```JAVA
class Solution {
    public TreeNode removeLeafNodes(TreeNode root, int target) {
        if (root == null){
            return null;
        }
        //如果需要删除赋值null，否则返回原节点
        root.left = removeLeafNodes(root.left, target);
        root.right = removeLeafNodes(root.right, target);
        if (root.val == target && root.left == null && root.right == null){
            return null;
        }else {
            return root;
        }
    }
}
```
